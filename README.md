## 📂 Estructura / Structure / Struktur

```
src/
├── components/     # Componentes Astro/UI
├── i18n/           # Traducciones ES/EN/DE
├── layouts/        # Layouts principales
├── pages/          # Páginas Astro
├── scripts/        # Scripts cliente (theme, i18n, animations)

```
